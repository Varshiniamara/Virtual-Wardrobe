# Screenshots & Final Output

**Mandatory for Hackathon Submission**

Please upload screenshots of your working application here. 
The judges need to see the "Final Output" as part of the repository requirements.

## Checklist
- [ ] **Home Page**: The landing page of your Virtual Wardrobe.
- [ ] **Clothing List**: View of the wardrobe items.
- [ ] **Outfit Creation**: Visual of the outfit combining feature.
- [ ] **Mobile View**: Screenshot of the app on a mobile width (responsive check).

*Tip: Name your files clearly, e.g., `home-page.png`, `mobile-view.jpg`.*
